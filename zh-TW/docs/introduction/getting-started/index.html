入門

在開始使用TouchGFX前，需要安裝最新版的TouchGFX，這將在安裝部分中介紹。

完成該操作後，您就可以開始第一個TouchGFX專案。 強烈建議您閱讀本章節，以熟悉最基本的TouchGFX概念。 最基礎的章節為開發簡介，此部分將為您介紹完整的TouchGFX專案所需的軟硬體元件，以及創建它們所需的操作及工具。 該章節還介紹了如何使用預製元件開始快速原型設計。

TouchGFXDesigner 快速入門#

TouchGFX Board Setups + Examples and Demos#

如果您渴望開始體驗TouchGFX，請試著一個UI範例專案，並在STM32評估套件上執行該專案，甚至可以啟動TouchGFXDesigner。

Creating an application with a TouchGFX Board Setup and an Example

On the lobby in TouchGFX Designer you can select from a wide range of examples and demos to start from. You can even combine this with board specific code for a range of STM32 Evaluation Kits, so you can have an UI running on the embedded hardware right away.

如果沒有STM32評估套件，您也可以只在PC上運行TouchGFX應用程式。 如果您希望根據操作指南完成第一步，則可以查看教程章節。 這裡將向您展示如何啟動並運行專案，最終完成一個相當複雜的UI應用程式。

Board Specific Demos#

Aside from being able to combine a TouchGFX Board Setup with an Example or a Demo, TouchGFX Designer also includes some out of the box demos for specific STM32 Evaluation Kits. These are loacted in the Demos section under Board Specfic Demo.

下一步#

此時，建議您流覽文件並閱讀相關章節。 這將使您對文件有一個大致的瞭解，從而讓您知道從哪裡查尋更多資訊，並瞭解TouchGFX關鍵概念。

主要章節包括：

  • 基本概念: 本章為本文件的其餘章節提供背景知識。 其中將介紹稍後要引用的所有關鍵圖形概念。 根據您當前的知識以及您將進行的TouchGFX開發操作，您可能想先跳過某些章節，並在以後不瞭解某些細節的情況下回過頭來查看這些章節。
  • 研發t: 本章為本文件的主要章節。 它解釋了TouchGFX專案的結構以及整個專案開發週期中涉及的工作流程和工具。 工作流程中的每個步驟都有各自的章節，並對其進行了詳細描述。 您可以在其中找到有關如何使您的應用程式在硬體上運行，以及如何進行UI開發的全部所需知識，包括對TouchGFXFramework中的所有功能和元件的描述。
