Q15

Defines a "floating point number" with 15 bits reserved for the fractional part of the decimal number. Q15 is only used for sine/cosine and for intermediate calculations when multiplying.

Q5*Q5=Q10, Q10/Q5=Q5, ...

See: Q5, Q10
